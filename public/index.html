為創業者建構完善的創業

環境

推動共學計畫

建構一個以「創業者」為核心的學習社群，透過導師與創業者及創業者間多元的學習分享方式，協助創業者及核心團隊在新創企業不同發展階段建立或強化企業經營及管理成長的能力。

Read more
經營創業社群

透過非正式的活動，聚會以及相關資訊的報導分享，加強導師與創業者及創業者相互間進一步的認識與交流，凝聚成員對社群的歸屬感，進而建立一個相互信任的創業社群。

Read more
探索共創模式

建構一個新創企業與政府、企業及創業者間共創的平台。透過新創企業與政府深度的對話，形成改善台灣創業生態系統的政策。推動新創企業與中大型企業的策略合作，探索共創價值的模式。

Read more
鏈結夥伴關係

選擇有共同理念的合作夥伴，包括中央及地方政府、國內外中大型企業，專業服務機構及投資金融機構，透過深度的交流及合作，建立協助新創企業的合作機制。
